## 1. デザインコンセプト

### 1.1 デザインテーマ
- **シンプル & クリーン**: 旅の記録に集中できるミニマルなUI
- **直感的**: 誰でもすぐに使える操作性
- **視覚的**: 地図・写真を中心とした視覚的な体験
- **モダン**: 最新のモバイルアプリトレンドを反映

### 1.2 カラーパレット

```
プライマリカラー:
  - Main: #4A90E2 (青) - 信頼感、旅の広がり
  - Light: #7AB2F5
  - Dark: #2E6BAE

セカンダリカラー:
  - Main: #50C878 (緑) - 訪問済み、達成感
  - Light: #7FD99A
  - Dark: #3A9B5C

ニュートラルカラー:
  - Background: #FFFFFF (白)
  - Surface: #F5F7FA (薄いグレー)
  - Text Primary: #1A1A1A (黒)
  - Text Secondary: #6B7280 (グレー)
  - Divider: #E5E7EB (境界線)

アクセントカラー:
  - Success: #10B981 (成功)
  - Warning: #F59E0B (警告)
  - Error: #EF4444 (エラー)
  - Info: #3B82F6 (情報)

訪問状態:
  - Unvisited: #9CA3AF (未訪問 - グレー)
  - Visited 1st: #4A90E2 (1回目 - 青)
  - Visited 2nd: #8B5CF6 (2回目 - 紫)
  - Visited 3+: #F59E0B (3回以上 - オレンジ)
```

### 1.3 タイポグラフィ

```
フォントファミリー:
  - iOS: San Francisco (システムデフォルト)
  - Android: Roboto (システムデフォルト)
  - 日本語: Noto Sans JP (fallback)

フォントサイズ:
  - H1: 28px (太字) - 画面タイトル
  - H2: 24px (太字) - セクションタイトル
  - H3: 20px (太字) - サブタイトル
  - Body: 16px (通常) - 本文
  - Caption: 14px (通常) - キャプション
  - Small: 12px (通常) - 小さいテキスト

行間:
  - 見出し: 1.2
  - 本文: 1.5
```

### 1.4 スペーシング

```
パディング/マージン:
  - XS: 4px
  - SM: 8px
  - MD: 16px
  - LG: 24px
  - XL: 32px
  - XXL: 48px
```

### 3.1 地図画面（Map Screen）

#### レイアウト

```
┌────────────────────────────────────────┐
│  🔍 検索                    🎯 現在地  │ ← ヘッダー
├────────────────────────────────────────┤
│                                        │
│         🗾 地図エリア                   │
│                                        │
│   📍 📍 📍 📍 📍                      │
│      📍   📍  📍  📍                  │
│   📍      📍      📍                  │
│      📍      📍      📍               │
│                                        │
│                                        │
│                                        │
│                                        │
├────────────────────────────────────────┤
│   [地図] [スレッド] [投稿] [カレンダー] [マイ]  │
└────────────────────────────────────────┘
```

#### 機能要素

**ヘッダー:**
- 検索ボックス（街名検索）
- 現在地ボタン
- フィルターボタン（有料版）：自分のみ / 友達含む

**地図エリア:**
- react-native-mapsで実装
- 街マーカー（訪問状態でアイコン変化）
- マーカータップで街詳細ボトムシート表示

**街マーカーのアイコンデザイン:**
```
自分の訪問状態:
  - 未訪問: 📍 (グレーのピン)
  - 1回訪問: ✅ (青いチェックマーク)
  - 2回訪問: ⭐️ (紫のスター)
  - 3回以上: 🏆 (オレンジのトロフィー)

友達の訪問状態（有料版）:
  - 友達のみ訪問: 👥 (半透明の人型アイコン)
  - 自分と友達両方訪問: 複合アイコン（自分のアイコン + 小さい👥バッジ）

実装詳細:
  - react-native-maps の <Marker> コンポーネントでカスタムアイコン使用
  - SVGまたはPNG画像でアイコン作成
  - ズームレベルに応じてアイコンサイズ調整
```

  variant: 'primary' | 'secondary' | 'outline' | 'ghost';
  size: 'sm' | 'md' | 'lg';
