# マップ一覧表示の種類

| コンポーネント | 用途                                           |
| -------------- | ---------------------------------------------- |
| MapDisplayCard | 発見タブホームの横スクロールセクション         |
| MapGridCard    | タグページのグリッド表示                       |
| MapListCard    | 一覧ページ（人気/ピックアップ/履歴）の縦リスト |
| MapCompactCard | マイページ/コレクションのコンパクトリスト      |
