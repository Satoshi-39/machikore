# E2E テストシナリオ一覧

Maestro を使用したE2Eテストのシナリオ一覧。
ユーザージャーニーベースで整理し、チェックボックスで実装状況を管理する。

## 認証・オンボーディング

- [x] アプリ起動時にオンボーディングを通過してホーム画面が表示される (`01_app_launch`)
- [x] 未ログイン状態で作成ボタンをタップすると認証モーダルが表示される (`04_create_requires_auth`)
- [x] メールOTPでログインできる (`10_auth_email_login_with_onboarding`)
- [ ] OAuth2（Google / Apple）でログインできる ⚠️ 手動テストのみ
- [x] ログイン後にオンボーディング（ユーザー名設定等）が表示される (`10_auth_email_login_with_onboarding`)
- [x] オンボーディング後にチュートリアルが表示・完了できる (`11_auth_tutorial_completion`)
- [x] ログアウトできる (`12_auth_logout`)

## ホーム（フィード閲覧）

- [x] おすすめタブでマップ一覧が表示される (`05_home_feed_recommend`)
- [x] フォロー中タブでフォロー中ユーザーのマップが表示される (`06_home_feed_following`)
- [x] マップカードをタップしてマップ記事画面に遷移する (`07_home_feed_map_card_navigation`)
- [x] マップカードのマップアイコンをタップしてマップ画面に遷移する (`07_home_feed_map_card_navigation`)
- [x] スポットカードをタップしてスポット記事画面に遷移する (`08_home_feed_spot_card_navigation`)
- [x] フィードを下にスクロールして追加データが読み込まれる（無限スクロール） (`09_home_feed_infinite_scroll`)
